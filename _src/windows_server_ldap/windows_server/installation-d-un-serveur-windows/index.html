# Installation d'un serveur windows

Dans 
Windows server
●
Publié 2021-11-12

Un Serveur windows permet à une entreprise de gérer un ou plusieurs rôles.

Le rôle est un terme propre a windows server qui désigne une fonctionnalité ajoutée après son installation.

AD DS DNS DHCP
Active directory (contrôleur de domaine) Résolution de noms Distribution de la configuration réseau (IP, DNS, Gateway, PXE...)
Services de stockage (DFS)
WSUS
WDS
Hyper-V
RDS

Liens logiques de stockage et réplication

déploiement de mises à jour

déploiement d'images systèmes

Virtualisation de services
